# Understanding HTML Elements

- HTML is a "Markup Language"
- `<h1>Hello World!</h1>` HTML Element is made up of HTML Tags and Element Content
- `<h1></h1>` Opening Tag and Closing Tag
- `h1` Element Name
- `Hello World!` Element Content
